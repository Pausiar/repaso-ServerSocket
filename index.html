Sin hilos

Completa el servidor que escucha en el puerto 8080, acepta conexiones, espera 5 segundos y envía "Hola, mundo". Las conexiones se atienden en serie.

    }

Ejercicio 2 — Servidor con hilos

Con Thread

Modifica el servidor anterior para que cada conexión se maneje en un hilo independiente. Así, varios clientes pueden conectarse a la vez sin esperar al anterior.

import java.net.*;
import java.io.*;

public class ServerThread {
    public static void main(String[] args) throws Exception {

        ServerSocket serverSocket = new ServerSocket(8080);

        while (true) {

            Socket socket = serverSocket.();

            // 2. Crear e iniciar un Thread con lambda
            new (() -> {
                try {
                    Thread.(5000);
                    PrintWriter writer = new PrintWriter(socket.getOutputStream(), true);
                    writer.println("Hola, mundo");
                    socket.close();
                } catch (Exception e) { e.printStackTrace(); }
            }).();  // 3. Arrancar el hilo
        }
    }
}

Ejercicio 3a — Chat: Servidor

Lectura + Escritura

Servidor de chat: acepta una conexión, lee mensajes con readLine() e imprime por consola. Si recibe "PING" responde "PONG" automáticamente; si no, hace eco: "Echo: " + línea.

ChatServer.java
import java.net.*;
import java.io.*;

public class ChatServer {
    public static void main(String[] args) throws Exception {

        ServerSocket serverSocket = new ServerSocket(7000);
        System.out.println("Servidor esperando...");

        // 1. Aceptar una conexión
        Socket socket = serverSocket.();

        // 2. Abrir lector de datos del socket
        BufferedReader reader = new (
            new InputStreamReader(socket.()));

        // 3. Abrir escritor hacia el cliente
        PrintWriter writer = new PrintWriter(socket.(), true);

        // 4. Leer mensajes en bucle
        String line;
        while ((line = reader.()) != null) {
            System.out.println("Recibido: " + line);
            if (line.equals("PING")) {
                writer.("PONG");
            } else {
                writer.println("Echo: " + line);
            }
        }
        reader.close(); writer.close(); socket.close(); serverSocket.close();
    }
}

Sin hilos

Escribe un servidor completo en el puerto 9000 que:

2) Envíe la fecha y hora actual (LocalDateTime.now()) al cliente.

Con Thread

Escribe un servidor completo en el puerto 9001 que:
1) Acepte conexiones en hilo por cada cliente.
2) Lea una línea del cliente con readLine().
3) La devuelva en mayúsculas con toUpperCase().
4) Cierre el socket del cliente en el hilo.

🗒️ Checklist — ¿Has incluido…?
EchoUpperServer.java — solución
import java.net.*;
import java.io.*;

public class EchoUpperServer {
    public static void main(String[] args) throws Exception {
        ServerSocket serverSocket = new ServerSocket(9001);
        System.out.println("Servidor eco escuchando en 9001");

        while (true) {
            Socket socket = serverSocket.accept();

            new Thread(() -> {
                try {
                    BufferedReader reader = new BufferedReader(
                        new InputStreamReader(socket.getInputStream()));
                    PrintWriter writer = new PrintWriter(socket.getOutputStream(), true);

                    String line = reader.readLine();
                    if (line != null) {
                        writer.println(line.toUpperCase());
                    }
                    socket.close();
                } catch (Exception e) { e.printStackTrace(); }
            }).start();
        }
    }
}

Examen 1a — Conversor de temperaturas (Servidor)

Puerto 9090 · Multi-cliente

Servidor que escucha en el puerto 9090. Recibe una temperatura en °C del cliente y responde con su equivalente en °F (F = C × 9/5 + 32). El cliente puede enviar varias temperaturas seguidas; si envía salir, el servidor cierra esa conexión y vuelve a esperar a otro cliente. Atiende varios clientes a la vez con hilos.

🗒️ Checklist — ¿Has incluido…?
ConversionServidor.java — solución
import java.net.*;
import java.io.*;

public class ConversionServidor {
    public static void main(String[] args) throws Exception {
        ServerSocket serverSocket = new ServerSocket(9090);
        System.out.println("Servidor de conversión escuchando en 9090");

        while (true) {
            Socket socket = serverSocket.accept();

            new Thread(() -> {
                try {
                    BufferedReader reader = new BufferedReader(
                        new InputStreamReader(socket.getInputStream()));
                    PrintWriter writer = new PrintWriter(socket.getOutputStream(), true);

                    String linea;
                    while ((linea = reader.readLine()) != null) {
                        if (linea.equalsIgnoreCase("salir")) break;
                        try {
                            double c = Double.parseDouble(linea.trim());
                            double f = c * 9.0 / 5.0 + 32.0;
                            writer.println(f);
                        } catch (NumberFormatException e) {
                            writer.println("ERROR");
                        }
                    }
                    socket.close();
                } catch (Exception e) { e.printStackTrace(); }
            }).start();
        }
    }
}

Teclado + bucle

Cliente que se conecta a localhost:9090. Pide por teclado una temperatura en °C (o la palabra salir), la envía al servidor y muestra el resultado en °F. Repite hasta que el usuario escriba salir.

import java.net.*;
import java.io.*;
import java.util.Scanner;

public class ConversionCliente {
    public static void main(String[] args) throws Exception {

            new InputStreamReader(socket.getInputStream()));

            System.out.println("Resultado: " + resp + " °F");
        }

    }
}

Examen 2a — Adivina el número (Servidor)

Puerto 8080 · Partida por cliente

Servidor en el puerto 8080. Cada cliente que se conecta juega su propia partida en un hilo. Al conectarse, el servidor genera un número aleatorio entre 1 y 100 y envía "Adivina un número entre 1 y 100". Por cada intento responde:
· MAYOR si el secreto es mayor que el enviado.
· MENOR si el secreto es menor.
· ACERTASTE en X intentos al acertar (y cierra la conexión).

🗒️ Checklist — ¿Has incluido…?
Servidor.java — solución
import java.net.*;
import java.io.*;
import java.util.Random;

public class Servidor {
    public static void main(String[] args) throws Exception {
        ServerSocket serverSocket = new ServerSocket(8080);
        System.out.println("Servidor adivinanza escuchando en 8080");

        while (true) {
            Socket socket = serverSocket.accept();

            new Thread(() -> {
                try {
                    BufferedReader reader = new BufferedReader(
                        new InputStreamReader(socket.getInputStream()));
                    PrintWriter writer = new PrintWriter(socket.getOutputStream(), true);

                    int secreto = new Random().nextInt(100) + 1;
                    int intentos = 0;
                    writer.println("Adivina un número entre 1 y 100");

                    String linea;
                    while ((linea = reader.readLine()) != null) {
                        int n = Integer.parseInt(linea.trim());
                        intentos++;
                        if (n < secreto)      writer.println("MAYOR");
                        else if (n > secreto) writer.println("MENOR");
                        else {
                            writer.println("ACERTASTE en " + intentos + " intentos");
                            break;
                        }
                    }
                    socket.close();
                } catch (Exception e) { e.printStackTrace(); }
            }).start();
        }
    }
}

Examen 2b — Adivina el número (Cliente)

Teclado + bucle

Cliente que se conecta a localhost:8080. Muestra el mensaje inicial del servidor, pide un número por teclado, lo envía y muestra la respuesta. Termina cuando recibe una línea que empieza por ACERTASTE.

🗒️ Checklist — ¿Has incluido…?
Cliente.java — solución
import java.net.*;
import java.io.*;
import java.util.Scanner;

public class Cliente {
    public static void main(String[] args) throws Exception {
        Socket socket = new Socket("localhost", 8080);
        BufferedReader reader = new BufferedReader(
            new InputStreamReader(socket.getInputStream()));
        PrintWriter writer = new PrintWriter(socket.getOutputStream(), true);
        Scanner sc = new Scanner(System.in);

        System.out.println("Conectado al servidor.");
        System.out.println(reader.readLine()); // "Adivina un número entre 1 y 100"

        while (true) {
            System.out.print("Tu número: ");
            String entrada = sc.nextLine();
            writer.println(entrada);

            String resp = reader.readLine();
            if (resp == null) break;
            System.out.println(resp);
            if (resp.startsWith("ACERTASTE")) break;
        }
        socket.close();
    }
}
